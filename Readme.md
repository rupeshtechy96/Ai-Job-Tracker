# 🚀 AI Job Tracker

**AI Job Tracker** is a full-stack web application that helps users efficiently track and manage their job applications.  
It allows users to organize applications across different stages such as **Applied, Phone Screen, Interview, Offer, and Rejected**, while also providing **AI-powered job description parsing** to extract important job details automatically.

---

# ✨ Features

- 📋 Track job applications using a **Kanban-style dashboard**
- 🤖 **AI-powered Job Description Parser**
- 🧠 Generate **resume bullet suggestions**
- 🔐 **Secure authentication** with JWT
- 📊 Dashboard with **application statistics**
- 🧩 **Responsive modern UI**
- ☁️ Cloud database with **MongoDB Atlas**
- ⚡ Fast frontend built using **Vite + React**
